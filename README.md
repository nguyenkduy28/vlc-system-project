# VLC System Project

Graduation project for a Visible Light Communication system.

## Scope
This repository contains:
- Hardware design
- Firmware for STM32-based TX/RX
- PC software tools and GUI
- Test results and measurement data
- Project documentation

## Main modules
- Power management
- Optical transmitter
- Optical receiver
- ADC/comparator signal path
- STM32 firmware
- PC monitoring software

## Repository structure
- docs/: thesis, reports, block diagrams, datasheets
- hardware/: schematic, PCB, BOM, simulation, library
- firmware/: STM32 source code for TX/RX
- software/: PC tools, GUI, logger, analysis
- test/: BER, waveform, power, link validation
- production/: manufacturing outputs
